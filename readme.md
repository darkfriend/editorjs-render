Render block editor.js
=====

## Supported blocks
* code
* heading
* header
* image
* list
* paragraph
* delimiter
* checklist
* quote
* raw
* table
* attaches
* linkTool
* embed
* need more? Write me :)

## How Install

``` 
npm i editorjs-render -S
```

### import
```javascript
import blocksToHtml from 'editorjs-render';
```
or
```javascript
const blocksToHtml = require('editorjs-render');
```

## How to use?

```javascript
blocksToHtml(blocks);
```
